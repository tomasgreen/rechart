rechart

  • Dependency free
  • Responsive - Reduces data making the chart less cluttered at small sizes
  • Style with CSS
  • Tooltips adapted to touch-devices
  • 21 KB miminized including css (if that's considered small)
  • Customziable (well... it has settings)
Download Github Documentation

JS

rechart({
    element: document.getElementById('chart1'),
    dataset: [{
        data: randomData(randCount),
        color: 'white',
        label: 'Windows Phone'
    }],
    xAxis: xAxis
}, {
    ratio: 0.3,
    minHeight: 200
});

Styling

Try not to mess with the container. Recalculating the width (including padding) is pretty slow and makes the site sluggish when resizing.

Settings

The following settings can be used in the option parameter when calling new Reachart(). I'm pretty sure (no guarantees) you can change these at any point by calling "instance.settings.X = X", just don't forget to call instance.draw() afterwards.

Property Description
minHeight
number (default: 0)		 Minimun height of the chart container in pixles
maxHeight
number (default: Infinity)		 Maximum height of the chart container in pixles
ratio
number (default: undefined)		 The width/height ratio of the chart. This setting does not effect minHeight and maxHeight. If the ratio is undefined it will be set using the current height and width of the container at the point of creation.
contentOffset
number (default: 10)		 The offset from the svg-element to where the data is drawn.
tooltipOffset
number (default: 10)		 Offset from the mousepointer/tooltipline to where the tooltip will be drawn.
tickOffset
number (default: 10)		 Offset from the mousepointer/tooltipline to where the tooltip will be drawn.
touchDelay
number (default: 300)		 Number of milliseconds from touchstart to when the tooltip is shown.
lockScrollOnTouchPress
bool (default: true)		 Enables/Disables the page scroll when the tooltip has been shown. Will only be used on touch devices.
textSpacing
number (default: 4)		 Enables/Disables
bezier
bool (default: false)		 Enables/Disables bezier curves. It's off by default since bezier curves distorts data and can even go ut of bounds from the chart element (which won't look pretty). It will be automatically disabled if there are less than 3 points.
bezierDenominator
number (default: 9)		 Large number means small curve, small number means large curve. This is some kind of denominator that determines the size/tension of the curve. I stole the code from chartist so I don't really understand how it works...
circleRadius
number (default: 4)		 The radius of the circles in pixels.
circleRadiusHover
number (default: 8)		 The radius of the circles in pixels when hovering above a line related to a circle.
responsive
bool (default: true)		 Enable/Disable the responsivness.
reduceData
bool (default: true)		 Enable/Disable data reduction. The data reduction itself is based upon the minimum distance of the x-axis grid.
reduceXAxisLabel
bool (default: true)		 Enable/Disable reduction of the xAxis-label. The xAxis labels are reduced based on the size of the label so that they never overlap.
keepRatio
bool (default: true)		 Enable/Disable ratio. If set to false the height of the chart will stay the same-
click
function(label,data,index) (default: undefined)		 Event for when the user clicks on the chart. This returns the x-axis label, data and index.
xAxisGridFollowLabel
bool (default: false)		 Enable/Disable whether or not the grid distance should follow the distance of the label.
xAxisGridMinDistance
number (default: 30)		 Minimum distance of the x axis grid lines.
xAxisMinDistance
number (default: 20)		 The miminum distance in pixels from one x axis point to another.
yAxisMinDistance
number (default: 20)		 The miminum distance in pixels from one y axis grid line to the next
showXAxisLabel
bool (default: true)		 Enable/Disable the xAxis label.
showXAxisGrid
bool (default: true)		 Enable/Disable the xAxis grid.
showXAxisLine
bool (default: true)		 Enable/Disable the xAxis bottom line.
showYAxisLabel
bool (default: true)		 Enable/Disable the yAxis label.
showYAxisGrid
bool (default: true)		 Enable/Disable the yAxis grid.
showYAxisLine
bool (default: true)		 Enable/Disable drawing of the yAxis bottom line.
showArea
bool (default: true)		 Enable/Disable drawing of the area/path.
showLine
bool (default: true)		 Enable/Disable drawing of the line/polyline.
showCircle
bool (default: true)		 Enable/Disable drawing of the circles.
showTooltip
bool (default: true)		 Enable/Disable drawing of the tooltip. This will also disable any custom implementation of the tooltip events.
showXAxisGuideline
bool (default: true)		 Enable/Disable drawing of the hover/touch guideline.
sizeSmall
number (default: 768)		 The responsive width (in pixles) that changes the labels to size small.
sizeMedium
number (default: 992)		 The responsive width (in pixles) that changes the labels to size medium.
sizeLarge
number (default: 1200)		 The responsive width (in pixles) that changes the labels to size large.

Methods

There's only really one method worth mentioning (draw(), for anyone who's interested), but what the hell, I'll just put them all here.

Changing the execution order of the drawSomething-methds will change the order of the elements. This might, for example, put the tooltip below everything else resulting in non-functioning tooltips.

Property Description
void draw(bool force) Draws the charts. Force will make sure that the chart is drawn reglardless of responsive settings.
bool prepare(bool force) This method is executed when draw() is called. If it returns true the chart will be drawn (or redrawn). Force will make sure that the chart is drawn reglardless of responsive settings.
void drawArea() Draws the area under the chart
void drawLine() Draws the polyline, which is the acual line.
void drawCircle() Draws the circles indicating each data point of the chart.
void drawXAxisGrid() Draws x axis grid
void drawYAxisGrid() Draws y axis grid
void drawXAxisLine() Draws x axis line, which is the bottom line.
void drawYAxisLine() Draws y axis line, which is the line to the right.
void drawXAxisGridTick() Draws x axis grid tick, that's the small line next to the text on the x axis.
void drawYAxisGridTick() Draws y axis grid tick, that's the small line beneath to the text on the y axis.
void drawXAxisGuideline() Draws the x axis line you see when the mouse hovers a tooltip rect.
void removeXAxisGuideline() Removes the guideline and resets the highlighted circles.
void dataIndexFromEvent(MouseEvent originalEvent) Returns the index corresponding to the position of the mouse.
void drawTooltip(Number index) Draws the tooltip with the data derived from the appointed index.
void positionTooltip(Event originalEvent, Number index) This method positions the tooltip according to settings and/or mouse-position. This method is called upon tooltiplineMouseenter()
void move(MouseEvent originalEvent) This method is called when the mouse moves inside the area. No touch event yet...
void leave(MouseEvent originalEvent) This method is called when the mouse leaves the area (or a touch-event ends).
void click(MouseEvent originalEvent) A funtion executed when the tooltip rect is clicked.
Object createLabel(Object obj, Element element) Returns an object that's used when drawing the approrpiate x axis label. The element in question is a temporary element with the right styling that is used to calculate the width of the label.
